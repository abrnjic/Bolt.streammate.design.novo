React Component API — Logo.tsx

// Props
type LogoSize    = 'sm' | 'md' | 'lg'
type LogoVariant = 'full' | 'icon' | 'wordmark'
type LogoTheme   = 'color' | 'white' | 'mono'

// Usage examples
<Logo />                                          // full · md · color (default)
<Logo size="lg" />                                  // hero / splash
<Logo variant="icon" size="sm" />                  // icon only — compact nav
<Logo variant="wordmark" size="sm" />             // wordmark only — footer
<Logo theme="white" />                              // white accent (solid dark bg)
<Logo variant="icon" size="lg" theme="white" />    // loading spinner companion

Usage Rules

Do

  • Use on dark backgrounds (#060b18 or darker)
  • Maintain aspect ratio — never stretch
  • Use the React component in-app
  • Use SVG files for external / non-React contexts
  • Min clearspace = bracket height on all sides

Don't

  • Don't change #00d4ff bracket/letter color
  • Don't place on light backgrounds without dark wrapper
  • Don't use "S" alone without brackets
  • Don't add drop shadows, glows, or filters
  • Don't reorder "Stream"/"Mate" or swap font weights
  • Don't use icon below 28 px wide
